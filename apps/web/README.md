# GoalPulse Agent Web Dashboard

This is the React frontend for GoalPulse Agent.

The dashboard is built for the TxLINE Trading Tools and Agents track and displays autonomous sports market intelligence generated by the GoalPulse backend.

## Live Frontend

https://goalpulse-agent.vercel.app

## Backend API

https://goalpulse-agent-api.onrender.com

## Main Dashboard Features

- Live market board
- Normalized home, draw, and away odds
- Precise match status and clock labels
- Odds movement chart
- Signal Intelligence Panel
- TXODDS field context
- Field Pressure Index
- Field-backed vs market-only movement labels
- Reliability warnings
- Results Settlement Audit
- Score breakdown rows for H1, H2, total goals, corners, red cards, and yellow cards
- Replay audit demo
- Judge Demo Guide

## Local Development

Use the local API:

cd C:\Projects\goalpulse-agent\apps\web
npm.cmd install
$env:VITE_API_BASE_URL="http://localhost:4000"
npm.cmd run dev -- --host 127.0.0.1 --port 5175 --strictPort

Open:

http://127.0.0.1:5175

## Production API Configuration

The production frontend should point to:

https://goalpulse-agent-api.onrender.com

Do not deploy the frontend with localhost as VITE_API_BASE_URL.

## Build

cd C:\Projects\goalpulse-agent
npm.cmd --prefix apps\web run build

## Compliance Boundary

The web dashboard is for sports analytics and market intelligence only. It does not place wagers, custody funds, execute trades, or facilitate betting.
